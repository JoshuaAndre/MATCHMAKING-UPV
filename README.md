# MATCHMAKING-UPV

Plataforma de matchmaking basada en MMR para videojuegos, construida con microservicios.

## Servicios

- player-service
- matchmaking-service
- rating-service
- match-service

## Objetivo del MVP

- Registrar jugadores
- Gestionar colas de matchmaking
- Emparejar jugadores por MMR
- Registrar partidas
- Actualizar MMR al finalizar partidas
